Control order of selection

Although it works on most cases, order of selection is not guaranteed.

users .select((f) => ({ abc: f.name, def: f.id, })) .stringify(); # formatted by sql-formatter SELECT name AS \`abc\`, id AS \`def\` FROM \`users\`users .select((f) => ({ ["123"]: f.age, name: f.name, ["456"]: f.id, })) .stringify(); # formatted by sql-formatter SELECT age AS \`123\`, id AS \`456\`, name AS \`name\` FROM \`users\`

To achieve control of the selection order, append each item individually.

users .select((f) => ({ ["123"]: f.age, })) .appendSelect((f) => ({ name: f.name, })) .appendSelect((f) => ({ ["456"]: f.id, })) .stringify(); # formatted by sql-formatter SELECT age AS \`123\`, name AS \`name\`, id AS \`456\` FROM \`users\`
